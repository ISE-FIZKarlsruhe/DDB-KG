Metadata Provenance

The current DDB infrastructure is based on the Open Archival Information System Model (OAIS), a standard in library management systems. There are around 500 participating public institutions providing metadata in varying exchange formats. It is imperative that ingesting external data into the DDB- EDM format can be traced in the KG. The PROV Ontology (PROV-O) is utilized for this purpose. The CHO and original metadata are defined as instances of prov:Entity. Since “ingest" is single activity, there is no need to define prov:Activity. The metadata provider, DDB and the XSLT mapper are defined as prov:Agent.
